
Сайт по аренде мотоциклов и заказу мототуров.

Две главные задачи проекта это выбор мотоцикла, или другой мототехники, для самостоятельной аренды 
и бронирование группового мототура. 


--------
MVP
--------
У сайта есть 2 ветки: 
    1. аренда мотоцикла
    2. бронировать мототур

Попадая на сайт клиенту необходимо выбрать необходимую услугу. 

1. АРЕНДА МОТОЦИКЛА 

Пользователю предостовляется выбор города в котором доступен прокат. 
Пользователю необходимо выбрать даты на которые он планирует бронировать технику.
Исходя из этих критериев он может увидеть доступные предложения в виде карточек превью,
на которых будет обозначена модель мотоцикла, год выпуска, фотогрфия, стоимость за сутки проката. 

Под карточкой есть 2 кнопки: БЫСТРАЯ БРОНЬ, ПОДРОБНЕЕ. 
По нажатию на кнопку ПОДРОБНЕЕ открывается страница, где указаны тех. характеристики техники,
условия брони и другая нужная информация. Есть кнопка ЗАКАЗАТЬ.

По нажатию на кнопку БЫСТРАЯ БРОНЬ/ЗАКАЗАТЬ открывается страница с формой. Пользователь вводит свои данные. 
После введения корректных данных пользователь переходит на страницу оплаты картой. 

После успешной оплаты пользователя перебрасывает на страницу с подтверждением и с благодарностью, 
на странице есть кнопка "возврат на главную". Также пользователю отправляется email с информацией о 
его бронировании. 

2. БРОНИРОВАТЬ МОТОТУР

Пользователю предоставляются все мототуры которые есть в доступности. 
Вверху страницы есть фильтр по дате, городу, продолжительности. 

Мототуры выводятся в виде карточек по аналогии арендных предложений. Под карточкой есть кнопка "ПОДРОБНЕЕ",
после ее нажатия открывается страница мототура на которой есть фотографии предыдущих туров и подробности.
Есть доступные даты тура и есть кнопка "БРОНИРОВАТЬ".

По нажатию на кнопку "БРОНИРОВАТЬ" открывается таже страница, что и при заказе аренды байка. Процесс прохождения 
бронирования и оплаты идентичен. Возможно будут отличия в необходимых данных. 


--------------------------------------
После запуска MVP планируется:

1. Блог, с реализацией комментариев.
2. Личный кабинет для пользователей. 
3. Бизнес аккаунт для людей, которые хотят предоставлять в аренду мотоциклы и проводить туры. 
4. Для туров - подключение гугл-карт с обозначением туров на них. Фотографии привязанные к локации по маршруту. 
5. Возможность оставлять отзыв к туру.
